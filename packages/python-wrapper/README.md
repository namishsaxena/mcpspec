# mcpspec

Convenience wrapper for [mcpspec-dev](https://pypi.org/project/mcpspec-dev/).

```bash
pip install mcpspec
```

```python
from mcpspec import McpSpec
```

This package re-exports the public API from mcpspec-dev for a cleaner import experience.
See the [mcpspec-dev documentation](https://mcpspec.dev) for full usage.
